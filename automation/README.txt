# Attendance Automation

## Getting Started

1. Go to Client Arrivals Report in MindBody.

2. Select date range of attendances in MindBody.

3. Save to Excel file in "reports" folder.

4. Run batch file to run Attendance Automation program.

5. Select matching date range in Attendance Automation.

6. Program will display # of successes and failures at the end.

7. Open Excel file to view error messages and # of success/fails.

## Misc

* Student key/pairs are in file "student.properties"

* Cookies + Verification Tokens in "credentials.txt" (these values may EXPIRE)
